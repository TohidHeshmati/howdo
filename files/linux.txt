# --- WC, SORT, & UNIQ (Data Processing) ---
[TEXT] wc -l | Count number of lines in a file or output
[TEXT] wc -w | Count number of words in a file
[TEXT] sort | Sort lines alphabetically
[TEXT] sort -n | Sort numerically (correctly handles 1, 2, 10)
[TEXT] sort -r | Reverse the sort order (Z-A or 10-1)
[TEXT] sort -k [n] | Sort by the nth column (e.g., sort -k 2)
[TEXT] uniq | Remove adjacent duplicate lines (always sort first!)
[TEXT] uniq -c | Count occurrences of each line (shows frequency)
[TEXT] uniq -d | Show ONLY the lines that are duplicated
[TEXT] uniq -u | Show ONLY the lines that are unique (no duplicates)
[TEXT] sort | uniq -c | sort -nr | The "Golden Combo": Finds most frequent items

# --- GREP (Searching Text) ---
[TEXT] grep "pattern" [file] | Search for a string in a specific file
[TEXT] grep -i "pattern" | Search case-insensitive (ignores Capitalization)
[TEXT] grep -v "pattern" | Invert match: show lines that do NOT contain pattern
[TEXT] grep -r "pattern" . | Recursive search: find pattern in all files in current folder
[TEXT] grep -l "pattern" * | List only the filenames that contain the pattern
[TEXT] grep -n "pattern" | Show line numbers alongside the matches
[TEXT] grep -E "A|B" | Extended Regex: search for "A" OR "B"
[TEXT] grep -A 2 "pattern" | Show matches plus 2 lines AFTER the match
[TEXT] grep -B 2 "pattern" | Show matches plus 2 lines BEFORE the match
[TEXT] grep -C 2 "pattern" | Context: show 2 lines before AND after the match
